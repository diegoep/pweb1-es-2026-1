Tipos de Seletores CSS

1. Seletor de Elemento

p { color: #555; }

Seleciona TODOS os parágrafos da página.

2. Seletor de ID

#principal { background-color: #fff; }

Seleciona o elemento com id="principal" (único na página).

3. Seletor de Classe

.destaque { background-color: #fff3cd; }

Seleciona TODOS os elementos com class="destaque" (reutilizável).

Este elemento usa duas classes: "destaque" e "texto-azul"

4. Seletor Descendente

#principal p { font-size: 16px; }

Seleciona todos os <p> que estão dentro de #principal.
