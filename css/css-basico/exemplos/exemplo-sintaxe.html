Sintaxe CSS

Estrutura Básica de uma Regra CSS

seletor {
propriedade: valor;
propriedade: valor;
}

Componentes da Sintaxe

  • Seletor: Indica qual(is) elemento(s) HTML será(ão) estilizado(s)
  • Propriedade: Característica que será modificada (cor, tamanho, etc.)
  • Valor: O novo valor da propriedade
  • Declaração: O conjunto propriedade + valor
  • Bloco de Declarações: Tudo que está entre { }
  • Regra CSS: Seletor + Bloco de Declarações

Exemplo Prático

.exemplo-visual {
background: linear-gradient(135deg, #667eea, #764ba2);
color: white;
padding: 30px;
border-radius: 10px;
text-align: center;
}
Resultado Visual

Regras Importantes da Sintaxe

  1. Cada declaração deve terminar com ; (ponto e vírgula)
  2. O bloco de declarações fica entre { e }
  3. Espaços em branco são ignorados (use para melhor legibilidade)
  4. CSS não diferencia maiúsculas de minúsculas para propriedades
  5. Valores de cores, URLs e outros podem ser case-sensitive
