Herança CSS

Herança é o mecanismo pelo qual elementos filhos herdam propriedades CSS de seus elementos pais.

Propriedades que são Herdadas vs Não Herdadas

Propriedade Herda? Categoria
color ✓ Sim Texto
font-family ✓ Sim Texto
font-size ✓ Sim Texto
line-height ✓ Sim Texto
text-align ✓ Sim Texto
background-color ✗ Não Visual
border ✗ Não Box Model
padding ✗ Não Box Model
margin ✗ Não Box Model
width / height ✗ Não Dimensões

Exemplo 1: Herança em Ação

Elemento PAI (color: white, font-size: 18px)
Elemento FILHO (herda color e font-size do pai)
Elemento NETO (herda color e font-size do pai e filho)

Exemplo 2: Propriedades NÃO Herdadas

Elemento PAI (background-color: #e74c3c, border: 5px solid)
Elemento FILHO (NÃO herda background-color nem border do pai)
💡 Regra Geral: Propriedades relacionadas a texto geralmente são herdadas. Propriedades relacionadas ao box model (layout) geralmente NÃO são herdadas.
