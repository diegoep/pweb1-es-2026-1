Especificidade CSS

A especificidade determina qual regra CSS será aplicada quando múltiplas regras selecionam o mesmo elemento.

Cálculo da Especificidade

Seletor Inline IDs Classes Elementos Especificidade
p 0 0 0 1 0,0,0,1
.texto 0 0 1 0 0,0,1,0
#especial 0 1 0 0 0,1,0,0
p.texto 0 0 1 1 0,0,1,1
#especial.texto 0 1 1 0 0,1,1,0
style="..." 1 0 0 0 1,0,0,0

Exemplos Visuais

Este parágrafo é cinza (especificidade: 0,0,0,1)

Este parágrafo é laranja porque p.texto (0,0,1,1) vence .texto (0,0,1,0)

Este parágrafo é verde porque #especial (0,1,0,0) vence p (0,0,0,1)

Este é roxo porque #especial.texto (0,1,1,0) tem maior especificidade

Regra de Ouro: Quanto maior a especificidade, maior a prioridade!
Inline (1,0,0,0) > ID (0,1,0,0) > Classe (0,0,1,0) > Elemento (0,0,0,1)
